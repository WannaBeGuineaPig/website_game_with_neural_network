### Введение:
Проект Deal with Pixel это веб-игра в которой лобби игроков рисуют на своём полотне заданную тематику. По истечению времени игроки смотрят все работы и игра показывает процент схожести на оригинал тематики. Побеждает игрок у которого больще всего процент схожести. Процент схожести определяет собственная нейросеть.

## Описание коммитов

|----------|-----------------------------------------------------------------|
| docs	   | Обновление документации                                         |
| feat	   | Добавление нового функционала                                   |
